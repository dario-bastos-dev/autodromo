# Autódromo

## Como executar

Para compilar e executar código, basta rodar `make run` no terminal.

**Obs:** Para o Windows, abra o terminal do **wls** ou do **git bash**, se não dará erro ao na etapa de `SOURCES` do `Makefile`.

## Como trabalhar

Crie uma nova branch com o nome do que será feito e trabalhar em cima dela. Após finalizar, realizar um push e solicitar uma **pull request** dentro do github.
Ex:`git switch -c gerar-logs`
